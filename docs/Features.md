# Features

## Event Impact Prediction
Forecast traffic demand caused by planned and unplanned events.

## Congestion Risk Detection
Classify locations into Low, Medium and High congestion risk categories.

## Heatmap Visualization
Display affected zones using interactive maps.

## Resource Allocation
Recommend police deployment and barricading requirements.

## Diversion Planning
Suggest alternate routes to reduce congestion.

## Post Event Analysis
Compare actual and predicted traffic conditions.
